# 课程与教材总表

如果你希望系统地啃透一本书，而不是仅仅看讲义，以下是几本被全球顶尖高校经济学与计算机系广泛采用的“圣经”级教材。

## 1. 绝对的入门首选 (For Absolute Beginners)

- **《策略思维》（Thinking Strategically）**
  - **作者**：Avinash K. Dixit / Barry J. Nalebuff
  - **特点**：几乎没有数学公式的纯科普读物，但极富洞察力。它会教你在日常生活中运用“承诺”、“威胁”和“逆向归纳”。作为睡前读物极佳。

- **《博弈论与经济模型》（Game Theory for Applied Economists）**
  - **作者**：Robert Gibbons
  - **特点**：被誉为“吉本斯小黄书”。极薄，极精炼。分为静态完全、动态完全、静态不完全、动态不完全四大块，是无数经济学本科生和研究生的启蒙红宝书。

## 2. 严谨的标准教材 (Standard Textbooks)

- **《博弈论导论》（An Introduction to Game Theory）**
  - **作者**：Martin J. Osborne
  - **特点**：讲解极尽细致，非常注重严谨的逻辑定义，例子丰富。

- **《博弈论导论》（Game Theory: An Introduction）**
  - **作者**：Steven Tadelis
  - **特点**：近年来极受欢迎的新教材。完美地结合了直觉解释与严谨的数学推导，每一章的逻辑过度极其丝滑，还包含了机制设计等相对现代的内容。**本仓库的大量架构即参考此书**。

## 3. 高阶与研究导向 (Advanced & Graduate Level)

- **《博弈论》（Game Theory）**
  - **作者**：Drew Fudenberg / Jean Tirole
  - **特点**：大名鼎鼎的“F&T”。厚如砖头，是研究生的标准参考书。内容全面，包含极大量的数学推演。如果某篇论文引用了基础定理，大概率会引用这本书。

- **《微观经济学原理》（Microeconomic Theory）**
  - **作者**：Andreu Mas-Colell / Michael D. Whinston / Jerry R. Green (MWG)
  - **特点**：微观经济学最高圣经。其博弈论部分虽然篇幅不占主导，但定义之严苛、视角之宏大无出其右。

## 4. 交叉学科教材 (Interdisciplinary)

- **《多智能体系统》（Multiagent Systems）**
  - **作者**：Yoav Shoham / Kevin Leyton-Brown
  - **特点**：计算机科学视角的博弈论。重点探讨算法、复杂度和人工智能交互。

- **《算法博弈论》（Algorithmic Game Theory）**
  - **作者**：Nisan / Roughgarden / Tardos / Vazirani
  - **特点**：了解互联网广告竞价机制、网络拥堵和算法设计的必读权威合集。
