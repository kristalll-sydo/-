# 01-如何使用本仓库

这个仓库现在不是单一层级的“笔记堆”，而是一个分层学习系统。你可以把它理解成四套互相配合的工具。

## 四层阅读法

### 第一层：课程入口

你用它解决“从哪开始”的问题。

推荐文件：

- [00-学习路线总览](00-学习路线总览.md)
- [02-课程地图与先修关系](02-课程地图与先修关系.md)
- [03-复盘方法与输出模板](03-复盘方法与输出模板.md)

### 第二层：原章节层

你用它解决“这一章严谨地讲了什么”的问题。

这里保留了原有的 `01-06` 章节结构，重点是：

- 学术定义
- 数学表达
- 经典模型
- 论文与课程链接

如果你已经有基础，或者正在修课，这一层会很高效。

### 第三层：理解性讲义层

你用它解决“我知道这个词，但我其实没真正想明白”的问题。

讲义层强调：

- 概念为什么提出
- 背后的现实困境是什么
- 为什么公式会长成这样
- 常见误解是什么
- 怎么用生活化或商业案例解释它

入口文件：

- [00-学习路线总览](00-学习路线总览.md)

### 第四层：图示、练习、案例层

你用它解决“怎么记住、怎么比较、怎么应用”的问题。

- `10-Assets`：看结构与信息图
- `11-Exercises`：查掌握程度
- `12-Case-Studies`：练建模与分析

## 建议的实际阅读动作

### 如果你是第一次学一章

1. 先看这一章要解决什么问题
2. 再看直觉解释和图示
3. 然后看数学表达中的变量、约束、目标
4. 最后用例子和练习检查自己是否真的理解

### 如果你已经学过，但总觉得不扎实

1. 先去看对应的理解性讲义
2. 再回原章节看公式和定义
3. 最后做一题“辨析题”或“案例题”

### 如果你正在准备考试

1. 用原章节快速过一轮
2. 用图示库建立知识结构
3. 用练习库做高频题型
4. 用复盘模板补漏洞

## 现在每章建议按这个顺序学习

`学习目标 -> 核心问题 -> 直觉理解 -> 数学表达 -> 经典例子 -> 图示说明 -> 易错点 -> 小结 -> 练习`

原仓库已经覆盖了其中的大部分骨架。本次升级重点补强了三个过去最容易断掉的环节：

- 直觉层不够连续
- 图示层不够稳定
- 练习与复盘层较弱

## 如何判断自己有没有真正学懂

读完一章后，不要只问“我会不会算”，还要问：

- 我能不能不用术语，把这个模型讲给外行人听？
- 我能不能说出它和相邻模型的差别？
- 我能不能解释这个均衡为什么成立，而不是只会写答案？
- 我能不能举一个现实问题，把它翻译成这个模型？

如果这四个问题里有两个答不上来，就建议回到讲义层和图示层再走一遍。

## 最后一个建议

把仓库当作“自己的课程工作台”，而不只是资料库。你完全可以：

- 在每章末尾写自己的反例和问题
- 为每个核心概念建立双链
- 把真实课程作业、考试题和论文阅读笔记接进来

这样它就不再只是一个外部仓库，而会变成你的长期知识系统。
