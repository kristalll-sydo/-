# 05 - 论文精读模板 (Paper Reading Template)

阅读博弈论核心文献，与阅读普通的经验类社会科学文献完全不同。博弈论文献高度抽象且数学密集。为了防止在复杂的公式推导中迷失方向，请严格按照以下**“剥洋葱式”精读模板**拆解任何一篇高级文献。

## 第一层：洋葱表皮 (核心故事与直觉)

*不看任何公式，只看 Abstract, Introduction 和 Conclusion。*
1. **Research Question**: 这篇文章试图解释什么现实中的不合理现象？（例如：为什么二手车市场会崩溃？）
2. **Key Mechanism**: 作者提出的核心机制是什么？用一句话大白话解释。（例如：因为买家不知道车好坏，只愿出平均价，导致好车退出市场。）
3. **Contribution**: 它推翻了之前的什么理论？或者填补了什么空白？

## 第二层：洋葱果肉 (模型构建与设定)

*进入 Model Setup 部分，搞清楚博弈的规则。*
1. **Players**: 有哪些参与者？（例如：买家和卖家）
2. **Information Structure**: 谁知道什么，谁不知道什么？（例如：卖家知道车的质量，买家不知道，这是典型的不完全信息）
3. **Action Space**: 参与者可以做什么动作？（例如：卖家标价，买家决定买不买）
4. **Payoff Functions**: 参与者的收益函数是什么？（写出具体的效用公式 $U_i$）

## 第三层：洋葱核心 (均衡求解与数学证明)

*这是最难的部分，深入核心数学推导。*
1. **Equilibrium Concept**: 文章用的是什么均衡概念？（纳什均衡、子博弈完美均衡、完美贝叶斯均衡？）
2. **Main Theorem**: 核心定理的数学表述是什么？
3. **Proof Strategy**: 证明的核心逻辑是什么？（例如：运用反证法证明分离均衡不存在，运用包络定理求解一阶导数，运用不动点定理证明存在性）。
*注意：如果是极度复杂的推导，第一遍阅读可以暂时跳过 Proof 细节，只承认 Theorem 结果。*

## 第四层：回味 (扩展与局限性)

1. **Critical Assumptions**: 哪些假设是极端且不现实的？（例如：假设所有买家都是完全理性的风险中性者）。如果放宽这些假设，结论还会成立吗？
2. **Real-world Applications**: 这个模型还能套用在哪些其他现实场景中？（例如：柠檬市场模型还能用来解释医疗保险市场）。
